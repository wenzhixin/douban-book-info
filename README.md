# 买书豆瓣速查

在买书的时候自动生成豆瓣读书的链接，可供快速查看图书的评分等相关信息

___

支持网店：

* 京东商场
* 亚马逊中国
* 当当网
* 文轩网
* 北发图书网
* 中国图书网
* china-pub
* 淘书网
